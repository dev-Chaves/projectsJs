# 30 Projetos em JavaScript - Iniciante

Este repositório contém 30 projetos desenvolvidos em JavaScript, com o objetivo de estudar e praticar habilidades de programação. Os projetos abrangem desde conceitos básicos de JavaScript até o uso de frameworks e APIs. Todos eles foram criados com fins educativos para aprimorar minhas habilidades como desenvolvedor iniciante.

## Tecnologias Utilizadas

- **JavaScript (ES6+)**
- **Node.js**
- **Express.js** (alguns projetos)
- **APIs externas** (usadas em alguns projetos)
- **HTML5 & CSS3** (para interface e estilização)

## Objetivo

Este repositório tem como finalidade principal:

1. **Aprender os fundamentos do JavaScript**
2. **Praticar lógica de programação**
3. **Desenvolver pequenos projetos para consolidar o aprendizado**
4. **Experimentar o uso de frameworks e APIs no backend (Node.js)**
